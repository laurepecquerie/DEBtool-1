Syntax

dMd = isotope_flux (t, Md)

Description

Input

  • t: scalar with time
  • Md: 13-vector with values for res M_E, struc M_V, mat M_H, otol M_OD, otol M_OG, del_*E, del_*V

Output

  • dMd: 13-vector with changes in Md
