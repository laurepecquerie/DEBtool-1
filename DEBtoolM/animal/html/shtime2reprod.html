shtime2reprod

Plots reproduction rate as a function of time

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

shtime2reprod

Description

Plots the reproduction rate at scaled functional responses f = 1, 0.8, 0.6, .. as functions of the volumetric lengths. The lowest value of f still has an ultimate length larger than l_p, which depends on other parameter values. See pages 69 ff of the DEB-book.

Remarks

The script pars_animal might be edited to see effects of parameter values

Example of use

clear all; pars_animal; shtime2reprod
