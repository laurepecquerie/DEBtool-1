Time plots for 'animal'

Syntax

shtime_animal

Description

Time plots for 'animal'

  • shtime2length: length as function of time
  • shtime2reprod: reproduction rate as function of time
  • shtime2weight: weight as function of time
  • shtime2survival: survival probability as function of time

Remarks

The script pars_animal might be edited to see effects of parameter values. The time intervals that are shown correspond with lengths upto 0.99 times the ultimate length.

Example of use

clear all; pars_animal; shtime_animal
